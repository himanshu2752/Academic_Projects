Project_2.zip folder contains following files -

-Summary.pdf - It contains the project report
-Design.pdf - It contains the design 
-Theatre.java - It contains the code 
- Readme.txt
- movies.txt - given test file

1. Project was developed using eclipse ide on Java
2. Number of Customer can be changed by changing variable "noCustomers" (It is currently set to 50). and number of box office agents can be changed by changing variable noAgents(it is currently set to 2).


3. The Theatre.java can be exceuted by going to the folder containing my java and test file and then running the following commands- 
javac Theatre.java
java Theatre movies.txt
